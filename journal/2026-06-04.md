### 备用长图风格与手机可读性
- 现象：用户希望备用长图每次生成风格不同，并反馈正文在手机上太小。
- 处理：备用长图计划增加 4 套 run 级风格：classic、warm-card、teal-digest、amber-memo；同一次生成四张长图保持统一风格，不同 run 按 revision 轮换，并把背景、卡片底色、线条、编号和强调色接入主题。
- 处理：备用长图正文基础字号从 28px 提高到 40px，重点正文从 33px 提高到 44px，并减少每行字数、提高长图高度估算，避免放大后溢出。
- 检查：`npm.cmd run test:run`、`npm.cmd run lint`、`npm.cmd run build --cache C:\tmp\npm-cache` 均通过。
